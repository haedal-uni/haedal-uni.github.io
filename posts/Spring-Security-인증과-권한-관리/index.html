Spring Security: мқёмҰқкіј к¶Ңн•ң кҙҖлҰ¬

backкіј frontлҘј л¶„лҰ¬н•ҳл©ҙм„ң н•ҙлӢ№ к¶Ңн•ңм—җ н•ҙлӢ№н•ҳлҠ” api мЈјмҶҢлҘј backм—җ л§һкІҢ ліҖкІҪн–ҲлӢӨ.

к·ёлҰ¬кі  н•ҙлӢ№ мЈјмҶҢ мӨ‘ кҙҖлҰ¬мһҗл§Ң мӮ¬мҡ©н•  мҲҳ мһҲкІҢ к¶Ңн•ңмқ„ Securityм—җ м„Өм •н–ҲлӢӨ.


л¬ёл“қ мқҙ к¶Ңн•ңмқҖ лЎңк·ёмқён•ң мӮ¬мҡ©мһҗ м •ліҙлҘј м–»кё° мң„н•ҙ

SecurityContextHolder.getContext().setAuthentication(authentication);лЎң к°’мқ„ м ҖмһҘн•ҳл©ҙ

@AuthenticationPrincipal UserDetails userDetailsлҘј мһ‘м„ұн–Ҳмқ„ л•Ң к°’мқ„ к°Җм ёмҳӨлҠ” кІғмІҳлҹј

SecurityContextHolderм—җ м ҖмһҘлҗң мӮ¬мҡ©мһҗ м •ліҙлҘј мӮ¬мҡ©н•ҳлҠ”кұёк№Ң мғқк°Ғмқҙ л“Өм—ҲлӢӨ.



Spring Security

Spring SecurityлҠ” Java кё°л°ҳмқҳ м• н”ҢлҰ¬мјҖмқҙм…ҳм—җм„ң ліҙм•Ҳкіј мқёмҰқмқ„ мІҳлҰ¬н•ҳкё° мң„н•ң н”„л Ҳмһ„мӣҢнҒ¬лЎң

REST API л°Ҹ м„ңл№„мҠӨлҘј ліҙнҳён•ҳкё° мң„н•ң лӢӨм–‘н•ң ліҙм•Ҳ кё°лҠҘмқ„ м ңкіөн•ҳл©°

мЈјмҡ” кё°лҠҘмңјлЎң мқёмҰқ(Authentication)кіј мқёк°Җ(Authorization)к°Җ мһҲлӢӨ.



  • мқёмҰқ(Authentication) : н•ҙлӢ№ мӮ¬мҡ©мһҗк°Җ ліёмқёмқҙ л§һлҠ”м§Җ нҷ•мқён•ҳлҠ” кіјм •

    мқёмҰқмқҖ мӮ¬мҡ©мһҗмқҳ мӢ мӣҗмқ„ нҷ•мқён•ҳлҠ” кіјм •мңјлЎң, мӮ¬мҡ©мһҗк°Җ м ңкіөн•ң мһҗкІ© мҰқлӘ…(мҳҲ: idмҷҖ pw)мқ„ кІҖмҰқн•ҳм—¬ мӮ¬мҡ©мһҗлҘј мӢқлі„н•ңлӢӨ.



  • мқёк°Җ(Authorization) : н•ҙлӢ№ мӮ¬мҡ©мһҗк°Җ мҡ”мІӯн•ҳлҠ” мһҗмӣҗмқ„ мӢӨн–үн•  мҲҳ мһҲлҠ” к¶Ңн•ңмқҙ мһҲлҠ”к°ҖлҘј нҷ•мқён•ҳлҠ” кіјм •

    мқёк°ҖлҠ” мқёмҰқлҗң мӮ¬мҡ©мһҗм—җкІҢ нҠ№м • лҰ¬мҶҢмҠӨ лҳҗлҠ” кё°лҠҘм—җ лҢҖн•ң м•Ўм„ёмҠӨ к¶Ңн•ңмқ„ л¶Җм—¬н•ҳлҠ” кіјм •мқҙлӢӨ.


Spring SecurityлҠ” кё°ліём ҒмңјлЎң мқёмҰқ м Ҳм°ЁлҘј кұ°м№ң нӣ„м—җ мқёк°Җ м Ҳм°ЁлҘј м§„н–үн•ҳл©°,

мқёк°Җ кіјм •м—җм„ң н•ҙлӢ№ лҰ¬мҶҢмҠӨм—җ м ‘к·ј к¶Ңн•ңмқҙ мһҲлҠ”м§Җ нҷ•мқён•ҳкІҢ лҗңлӢӨ


Spring SecurityлҠ” м„ёл¶„нҷ”лҗң к¶Ңн•ң кҙҖлҰ¬лҘј м§Җмӣҗн•ҳм—¬, м—ӯн•  кё°л°ҳмқҳ м ‘к·ј м ңм–ҙлҘј кө¬нҳ„н•  мҲҳ мһҲлӢӨ.

мқҙлҘј нҶөн•ҙ м• н”ҢлҰ¬мјҖмқҙм…ҳмқҳ лҰ¬мҶҢмҠӨм—җ лҢҖн•ң м ‘к·ј м ңн•ңмқ„ м„Өм •н•  мҲҳ мһҲлӢӨ.




Spring SecurityлҠ” вҖҳмқёмҰқвҖҷкіј вҖҳк¶Ңн•ңвҖҷм—җ лҢҖн•ң л¶Җл¶„мқ„ Filter нқҗлҰ„м—җ л”°лқј мІҳлҰ¬н•ҳкі  мһҲлӢӨ.

FilterлҠ” Dispatcher ServletмңјлЎң к°Җкё° м „м—җ м Ғмҡ©лҗҳлҜҖлЎң к°ҖмһҘ лЁјм Җ URL мҡ”мІӯмқ„ л°ӣм§Җл§Ң, (мӣ№ м»Ён…Ңмқҙл„Ҳм—җм„ң кҙҖлҰ¬)

InterceptorлҠ” DispatcherмҷҖ Controller мӮ¬мқҙм—җ мң„м№ҳн•ңлӢӨлҠ” м җм—җм„ң м Ғмҡ© мӢңкё°мқҳ м°Ёмқҙк°Җ мһҲлӢӨ. (мҠӨн”„л§Ғ м»Ён…Ңмқҙл„Ҳм—җм„ң кҙҖлҰ¬)


Client (request) вҶ’ Filter вҶ’ DispatcherServlet вҶ’ Interceptor вҶ’ Controller

(мӢӨм ңлЎң Interceptorк°Җ ControllerлЎң мҡ”мІӯмқ„ мң„мһ„н•ҳлҠ” кІғмқҖ м•„лӢҳ, InterceptorлҘј кұ°міҗм„ң к°ҖлҠ” кІғ)





Spring Security м„Өм •

Spring Security м„Өм •мқҖ мЈјлЎң configure() л©”мҶҢл“ңлҘј мӮ¬мҡ©н•ҳм—¬ мқҙлЈЁм–ҙм§„лӢӨ.

м—¬кё°м„ң URL кІҪлЎңм—җ лҢҖн•ң мқёк°Җ к·ңм№ҷмқ„ м„Өм •н•ҳкі , нҠ№м • URL кІҪлЎңлҘј н—Ҳмҡ©н•ҳкұ°лӮҳ мқёмҰқмқ„ мҡ”кө¬н•ҳлҠ” м„Өм •мқ„ н•  мҲҳ мһҲлӢӨ.

м„Өм •м„ӨлӘ…
http.authorizeRequests()URL кІҪлЎңм—җ лҢҖн•ң мқёк°Җ к·ңм№ҷмқ„ м„Өм •н•ңлӢӨ.
.antMatchers().permitAll()нҠ№м • URL кІҪлЎңлҘј мқёмҰқ м—Ҷмқҙ н—Ҳмҡ©н•ңлӢӨ.
.anyRequest().authenticated()лӘЁл“  мҡ”мІӯм—җ лҢҖн•ҙ мқёмҰқмқ„ мҡ”кө¬н•ңлӢӨ.
.hasRole()нҠ№м • м—ӯн• мқ„ к°Җм§„ мӮ¬мҡ©мһҗл§Ң м ‘к·јмқ„ н—Ҳмҡ©н•ңлӢӨ.
.formLogin()нҸј кё°л°ҳ лЎңк·ёмқёмқ„ нҷңм„ұнҷ”н•ңлӢӨ.
.loginPage()лЎңк·ёмқё нҺҳмқҙм§Җмқҳ кІҪлЎңлҘј м§Җм •н•ңлӢӨ.
.defaultSuccessUrl()лЎңк·ёмқё м„ұкіө нӣ„ мқҙлҸҷн•  кё°ліё URLмқ„ м„Өм •н•ңлӢӨ.
.logout()лЎңк·ём•„мӣғмқ„ мІҳлҰ¬н•ҳлҠ” м„Өм •мқ„ м¶”к°Җн•ңлӢӨ.
.logoutUrl()лЎңк·ём•„мӣғ URLмқ„ м§Җм •н•ңлӢӨ.
.logoutSuccessUrl()лЎңк·ём•„мӣғ м„ұкіө нӣ„ мқҙлҸҷн•  URLмқ„ м„Өм •н•ңлӢӨ.
.csrf()CSRF(Cross-Site Request Forgery) кіөкІ© л°©м–ҙ м„Өм •мқ„ нҷңм„ұнҷ”н•ңлӢӨ.
.sessionManagement()м„ём…ҳ кҙҖлҰ¬лҘј м„Өм •н•ңлӢӨ.
.sessionCreationPolicy()м„ём…ҳ мғқм„ұ м •мұ…мқ„ м„Өм •н•ңлӢӨ.

UserDetailsService(мӮ¬мҡ©мһҗ м •ліҙ мЎ°нҡҢ)

Spring SecurityлҠ” мӮ¬мҡ©мһҗ м •ліҙлҘј мЎ°нҡҢн•ҳкё° мң„н•ҙ UserDetailsServiceлҘј м ңкіөн•ңлӢӨ.

loadUserByUsername()м—җм„ң н•ҳлҠ” мқј

  • nicknameмқ„ к°Җм§Җкі  мӮ¬мҡ©мһҗ м •ліҙлҘј мЎ°нҡҢн•ҳм—¬ UserDetails к°қмІҙлЎң ліҖнҷҳн•ҳм—¬ л°ҳнҷҳ

  • мӮ¬мҡ©мһҗмқҳ Roleкіј к¶Ңн•ң(Privilege)мқ„ мЎ°нҡҢн•ҳм—¬, SimpleGrantedAuthority лӘ©лЎқмқ„ authoritiesм—җ м„ёнҢ…н•ңлӢӨ.

  • Authentication лӮҙл¶Җ principal к°қмІҙм—җ UserDetails к°қмІҙк°Җ м ҖмһҘлҗңлӢӨ.

  • Authentication лӮҙл¶Җ authorities к°қмІҙм—җ мӮ¬мҡ©мһҗмқҳ Roleкіј к¶Ңн•ң(Privilege) м •ліҙк°Җ м ҖмһҘлҗңлӢӨ.

  • UserDetailsм—җ authoritiesк°Җ м„ёнҢ…лҗҳм–ҙ мһҲм–ҙм•ј, APIлі„ roleмқҙлӮҳ к¶Ңн•ң мІҙнҒ¬лҘј м§„н–үн•  мҲҳ мһҲлӢӨ.





GrantedAuthority(к¶Ңн•ң кҙҖлҰ¬)

  • USER : мӮ¬мҡ©мһҗ м •ліҙлҘј нҸ¬н•Ён•ҳлҠ” entity к°қмІҙлЎң UserDetailsмқҳ кө¬нҳ„мІҙ

  • UserDetails : мқёмҰқлҗң н•өмӢ¬ мӮ¬мҡ©мһҗ м •ліҙ (к¶Ңн•ң, л№„л°ҖлІҲнҳё, мӮ¬мҡ©мһҗлӘ…, к°Ғмў… мғҒнғң)лҘј м ңкіөн•ҳкё° мң„н•ң interfaceмқҙлӢӨ.


Spring SecurityлҠ” мӮ¬мҡ©мһҗмқҳ к¶Ңн•ңмқ„ GrantedAuthorityлҘј нҶөн•ҙ кҙҖлҰ¬н•ңлӢӨ.

мқҙлҠ” нҳ„мһ¬ мӮ¬мҡ©мһҗ(Principal)к°Җ к°Җм§Җкі  мһҲлҠ” к¶Ңн•ңмқ„ мқҳлҜён•ҳл©°, мӮ¬мҡ©мһҗмқҳ Roleкіј к¶Ңн•ң(Privilege) м •ліҙлҘј м ңкіөн•ңлӢӨ.

SecurityContextHolder

SecurityContextHolderлҠ” SecurityContextлҘј ліҙкҙҖн•ҳлҠ” м ҖмһҘмҶҢлӢӨ.

мқҙлҘј нҶөн•ҙ нҳ„мһ¬ мӮ¬мҡ©мһҗмқҳ мқёмҰқ м •ліҙлҘј кҙҖлҰ¬н•ҳкі , Authentication мқёмҠӨн„ҙмҠӨлҘј м ҖмһҘн•ңлӢӨ.

*Authenticationм—җлҠ” principal, credentials, authoritiesк°Җ м ҖмһҘлҗңлӢӨ.



л”°лқјм„ң getAuthorities() л©”мҶҢл“ңлҘј нҶөн•ҳм—¬ к°Җм ёмҳӨлҠ” к¶Ңн•ңмқҖ

SecurityContextHolderм—җ м ҖмһҘлҗң мӮ¬мҡ©мһҗ м •ліҙлҘј нҷңмҡ©н•ҳм—¬ к°Җм ёмҳӨлҠ” кІғкіј лҸҷмқјн•ң кіөк°„м—җм„ң к°Җм ёмҳЁлӢӨкі  ліј мҲҳ мһҲлӢӨ.
