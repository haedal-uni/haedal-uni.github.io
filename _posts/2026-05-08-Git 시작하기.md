### repository 연결 
```bash
git init
git remote rm origin 
git remote add origin "주소 입력"
git pull origin main
```

<br><br><br>

### commit 하기 
```bash
git add .             
git commit -m "commit message"  
git push -u origin main # git push origin main 
```
*`-u` 사용 시 이후 git push만 입력 가능
