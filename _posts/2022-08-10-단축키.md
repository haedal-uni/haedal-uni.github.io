## IntelliJ 단축키 정리

### 기본 화면 조작

**Alt + 1** : Project 창 열기 / 닫기

**End → Shift + Home** : 현재 줄의 끝에서 처음까지 선택하기

<br>

### 리팩토링 / 코드 정리

**Ctrl + Alt + L** : 코드 자동 정렬, Reformat Code

**Ctrl + Alt + O** : 사용하지 않는 import 제거, Optimize Imports
※ 여기서 `O`는 숫자 0이 아니라 영문 O이다.

**Shift + F6** : 변수명, 메서드명, 클래스명 변경, Rename

<br>

### 검색 / 교체 / 이동

**Shift + Shift** : 전체 검색, Search Everywhere

**Ctrl + Shift + F** : 프로젝트 전체에서 검색, Find in Files / Find in Path

**Ctrl + Shift + R** : 프로젝트 전체에서 교체, Replace in Files / Replace in Path

**Ctrl + N** : 클래스 검색

<br>

### 코드 작성 보조

**Shift + Enter** : 현재 커서 위치와 상관없이 다음 줄 생성 
문장 끝으로 커서를 옮기지 않아도 바로 다음 줄로 이동할 수 있다.

**fori + Tab** : 기본 for문 자동완성

**iter + Tab** : 향상된 for문 자동완성

**Alt + Insert** : Generate 메뉴 열기    
생성자, Getter/Setter, `toString()`, Override 메서드 등을 자동 생성할 수 있다.
